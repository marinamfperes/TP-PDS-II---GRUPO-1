\doxysubsection{Detailed Description}
Funções auxiliares para conversão e formatação de datas e horários. 

Este arquivo define funções que convertem datas em strings no formato "{}dd/\+MM/yyyy HH\+:mm"{} para o tipo {\ttfamily time\+\_\+t} e vice-\/versa, usadas em todo o sistema de gerenciamento de tarefas.
